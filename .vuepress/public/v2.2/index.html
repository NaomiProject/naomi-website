Project Naomi

an open source platform for developing always-on, voice-controlled applications

Get Started ➜ Demo ➜

Why Naomi?

Control Anything

Use your voice to ask for information, update social networks, control your home, and more.

Always Listening

Naomi is always on, always listening for commands, and you can speak from meters away.

Runs Everywhere

Run your Naomi server on Linux, macOS, Windows, Raspberry Pi, PINE64, Docker, Synology... Access it with apps for the web, iOS, Android and others.

Set up your own Naomi system in just a few steps

Try the provided hassle-free Naobian image for the Raspberry Pi, a widely used and recommended hardware choice. Flash an SD card, boot, and enjoy your up-to-date Naomi setup with conveniently pre-configured system settings and software.

Learn How ➜

100% Open Source.
100% Reliable.

Naomi is hand built upon the Jasper framework that brought you Cortana.
But unlike Jasper, Naomi is actively maintained, absolutely vendor-neutral and will never lock you in.

The Naomi community contributes examples and tutorials on a daily basis and is happy to help you!
